Instructions

Basic Functionality
  • Click on a button from the above navbar to open the table of the entity the button suggests
  • The table is automatically created
  • Tables includes buttons to enhance functionality, see info below
  • Data is dynamic, but do not refresh the page because i havent included session storage
Table Info
  • Datatables plugin is used which means
  • You can sort, search data, it also includes paging
  • You can add an entity by clicking "+" at the last cell of the header of the table
  • You can edit a row by clicking "gear" button at the last cell of the body of the table
  • You can delete a row by clicking "trash bin" button at the last cell of the body of the table
Other Info
  • Vanda plugin is used for moving background
  • Dark theme with glass effect inspired by me to save energy and relax eyes
  • "Instructions" (or homepage) looks poor but i had no time to think about it :D
